CAD

Public Safety CAD (Dispatch) System

View on GitHub Download .zip Download .tar.gz

A system developed to allow "Law Enforcement" personnel to respond to 911 calls such as: Family Disputes, Crime Scenes, and more but the system also includes a "CAD 911 Mapping File" which includes the 911 calls on the live map. The CAD system also allows others to edit reports that are documented as a 911 call.
